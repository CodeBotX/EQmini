## Cài đặt

1. Clone repository:
```bash
git clone [https://github.com/CodeBotX/EQmini.git]
cd EQmini
```

2. Tạo và kích hoạt môi trường ảo:
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. Cài đặt các dependencies:
```bash
pip install -r requirements.txt
```
